# flsun-q5-klipper
Конфигурационные файлы Klipper для модифицированного принтера Flsun Q5.

Выполненные модификации:
1. Рельсы Hiwin
2. Direct Extruder Orbiter
3. Материнская плата Bigtreetech SKR Pico
4. Двигатели Trianglelab Geared 3:1
